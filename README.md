# Ariadne: An Educational Maze Solver & Algorithm Visualizer

Ariadne is a simple maze solver. It visualizes how some graph search algorithms such as **Breadth-First**, **Depth-First**, **Greedy Best-First**, **A\***, and **Dijkstra's** behave on mazes.

> [!NOTE]
> This project is a personal attempt to create an app using technologies I've never tried before. This means that the application will not be perfect!

## Project Overview & Background

### Who was Ariadne?

In Greek mythology, [**Ariadne**](https://en.wikipedia.org/wiki/Ariadne) was a Cretan princess, the daughter of [**King Minos**](https://en.wikipedia.org/wiki/Minos) of Crete. She is known for helping [**Theseus**](https://en.wikipedia.org/wiki/Theseus) navigate the [**Minotaur's Labyrinth**](https://en.wikipedia.org/wiki/Labyrinth). She gave him a ball of thread so he could find his way back out after slaying the creature.  

This project is named after her, because it was too good to pass up. She essentially solved (or helped solve) a maze, just like this app does with the use of graph search algorithms.  

She was the **first** maze solver!
